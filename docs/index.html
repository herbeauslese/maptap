► Wie funktioniert Glicko-2?

Jeder Spieler hat drei Werte: Rating (Startwert 1500, höher = besser), RD (Rating Deviation – Unsicherheit, Start 350, sinkt mit mehr Spielen), Volatility (Schwankungsneigung). Nach jedem Spieltag: jeder gegen jeden (höherer Score = Sieg). Wer gegen starke Spieler gewinnt steigt mehr. DNP = RD steigt leicht, Rating bleibt.
